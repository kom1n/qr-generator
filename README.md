# QR Code Generator

Jednoduchý generátor QR kódů vytvořený v čistém JavaScriptu.

## Funkce
- Generování QR kódů z textu nebo URL
- Export do PNG a SVG
- Nastavitelná velikost
- Žádné externí závislosti kromě QR knihovny
